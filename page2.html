Brioșe cu oreo

Ingrediente
Pentru briose:
15 biscuiti Oreo
200g unt/margarina
200g zahar
4 oua
150g faina
50g cacao
1-3 linguri lapte - daca este necesar
Pentru crema:
200g unt
500g zahar
6-8 Oreo, zdrobiti
2-4 lingurite lapte

Mod de preparare
1. Se incalzeste cuptorul la 180C si se tapeteaza o tava cu briose. Se pune un Oreo intreg in partea inferioara a fiecarei forme. Se lasa deoparte.
2. Se bat impreuna untul si zaharul pana cand capata o consistenta pufoasa si cremoasa.
3. Se adauga ouale, faina si cacao si se amesteca bine. Daca amestecul este putin rigid, se adauga 1 lingura de lapte pana cand ajunge la consistenta dorita.
4. Se pune amestecul in fiecare forma, deasupra biscuitelui Oreo si se coace in cuptor timp de 15-20 minute. Odata coapte, se lasa sa se raceasca pe un suport de sarma.
5. Pentru a face crema, se bate untul la temperatura camerei cu un mixer, apoi se adauga, treptat, zaharul pana cand este incorporat. Se adauga Oreo (macinat intr-un blender) si se amesteca timp de 3-5 minute pana cand consistenta devine pufoasa. Daca este prea rigid, se adauga laptele incet pana la consistenta dorita.
6. Folosind o duza de mari dimensiuni, se pune crema de unt peste briose. Se decoreaza cu biscuiti Oreo.
